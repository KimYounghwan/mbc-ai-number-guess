
# 🎯 Gemini's Number Quest

Gemini AI 마스터와 함께하는 스타일리시한 숫자 맞추기 게임입니다.

## ✨ 주요 기능
- **AI 게임 마스터**: Google Gemini API를 사용하여 매 추측마다 재치 있는 코멘트를 제공합니다.
- **스마트 힌트**: 정답이 너무 어려울 때 AI에게 신비로운 힌트를 요청할 수 있습니다.
- **모던 디자인**: Tailwind CSS와 애니메이션을 활용한 고급스러운 UI.
- **반응형 웹**: 모바일과 데스크톱 어디서든 완벽하게 작동합니다.

## 🚀 배포 방법 (Vercel)

1. 이 저장소를 본인의 GitHub로 **Fork** 하거나 파일을 업로드합니다.
2. [Vercel](https://vercel.com/)에 로그인하고 프로젝트를 Import 합니다.
3. **Environment Variables** 설정에서 다음을 추가합니다:
   - `API_KEY`: [Google AI Studio](https://aistudio.google.com/)에서 발급받은 API 키
4. 배포(Deploy) 버튼을 클릭하면 끝!

## 🛠 기술 스택
- HTML5, CSS3 (Tailwind CSS)
- JavaScript (Vanilla ES6+)
- Google Gemini API (`@google/genai`)
- Canvas Confetti
